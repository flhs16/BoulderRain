巨石雨 (BoulderRain) 插件使用说明

版本：v1.0.0
作者：天子 (TianZi)

一、功能介绍
巨石雨是一个为TShock服务器开发的Terraria插件，能够生成大量下落的巨石弹幕，为游戏增添趣味性和挑战性。

主要功能：
- 可配置的弹幕生成参数
- 支持追踪模式，可追踪怪物或玩家
- 灵活的权限控制
- 实时配置重载功能
- 完整的命令系统
- 详细的日志记录

二、安装方法
1. 确保已安装TShock服务器
2. 将RockRain.dll放入TShock的ServerPlugins目录
3. 启动服务器，插件会自动创建默认配置文件
4. 根据需要修改配置文件巨石雨.json

三、配置说明
配置文件位于TShock目录下的巨石雨.json，包含以下主要配置项：

1. 默认弹幕配置：
   - 弹幕ID：使用的弹幕类型ID，默认99
   - 下落速度：弹幕下落的基础速度，默认10.0
   - 伤害：弹幕造成的伤害值，默认50
   - 击退：弹幕的击退效果，默认5.0
   - 持续时间：效果持续的总时间(帧)，默认1800
   - 生成间隔：弹幕生成的时间间隔(帧)，默认10
   - 生成高度：弹幕生成的起始高度，默认600.0
   - 生成范围：弹幕生成的横向范围，默认1000.0
   - 追踪模式：是否启用追踪功能，默认false
   - 追踪目标：追踪的目标类型(0:无, 1:怪物, 2:玩家)，默认1
   - 追踪速度：追踪时的移动速度，默认5.0
   - 追踪范围：开始追踪的距离范围，默认500.0
   - 生命周期：弹幕的生命周期(帧)，默认300
   - 每帧更新次数：每帧生成的弹幕数量，默认0

2. 全局配置：
   - 调试模式：是否启用调试日志，默认false
   - 最大生成数量：同时存在的最大效果数量，默认10
   - 玩家加入自动开启：玩家加入时是否自动开启效果，默认false

四、命令说明
插件使用/rm（RockRain的缩写）作为主命令：

1. /rm help：显示帮助信息
2. /rm start：开始巨石雨效果（需要管理员权限）
3. /rm stop：停止巨石雨效果（需要管理员权限）
4. /rm set：查看当前弹幕属性设置（需要管理员权限）
5. /rm set <属性> <值>：设置弹幕属性（临时配置，需要管理员权限）
6. /rm reload：重载配置文件（需要管理员权限）

可设置的属性包括：
- speed：下落速度
- damage：伤害值
- knockback：击退效果
- duration：持续时间
- interval：生成间隔
- height：生成高度
- range：生成范围
- tracking：追踪模式
- target：追踪目标
- trackspeed：追踪速度
- trackrange：追踪范围
- lifetime：生命周期
- updates：每帧更新次数

五、示例用法
1. 开始巨石雨效果：
   /rm start

2. 设置弹幕伤害为100：
   /rm set damage 100

3. 启用追踪模式并设置追踪速度为8：
   /rm set tracking true
   /rm set trackspeed 8

4. 重载配置文件：
   /rm reload

六、注意事项
1. 使用过高的生成频率或数量可能会影响服务器性能
2. 请根据服务器配置和玩家数量合理调整参数
3. 调试模式会产生大量日志，请在调试完成后关闭
4. 插件在大型服务器上使用时建议进行性能测试
5. 需要rockrain.admin权限才能使用管理员命令

七、权限说明
- rockrain.admin：允许使用管理员命令
